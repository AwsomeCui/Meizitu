# 妹子图

#### 介绍
应朋友要求，写了一个宅男福利爬虫，利用`Scrapy`爬取妹子图网站(ituba.cc)，并下载到本地。

(⊙﹏⊙)额，请放心食用！

#### 软件架构
python 2.7
scrapy

#### 使用说明
在D盘新建文件夹meizi
```
git clone git@github.com:cc976655/Meizitu.git
cd Meizitu
python main.py
```
